## Public / Private Rule

Public = framework, summaries, redacted reports, governance policies.

Private = evidence, identities, financial records, contracts, court filings, tax records, protected family information.

## Core Operating Rule

GFO is designed to make life, contracts, publishing, consulting, token governance, education, and litigation-adjacent records transparent without exposing protected private information.

The public repository proves the framework.

The private data room preserves the evidence.
